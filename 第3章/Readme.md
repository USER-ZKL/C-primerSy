# 练习3.37

这个程序的意思是遍历字符数组，并将其中的字符进行输出。while循环中判断指针是否指向空字符，如果为空则说明到了字符串的末尾，就停止循环。

然而ca进行初始化时，没有空字符。因此该循环可能一直不会停止，直到遇到空字符为止。
